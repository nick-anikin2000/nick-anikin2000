Для того, чтобы добавить ветвь соедините между собой 2 узла, также можете использовать точки перегиба.
Для того, чтобы создать ветвь вам необходимо добавить не менее двух узлов, также можно использовать точки перегиба.
Для того чтобы добавить источник eds, выберите ветвь, на которой он будет распологаться, затем укажите значение напряжения и нажмите ОК.
Для того чтобы сформировать контур выберите 2 или более ветвей(branches) в дереве компонентов.
